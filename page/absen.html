Maenata Absen

  • masuk ke menu jadwal absen
  • buat jadwal pada form di sebelah kana
  • simpan dan aktifkan jadwal

    • untuk mengaktifkan dan menonaktifkan jadwal cukup metekan tombol aktif atau non aktif pada jadwal yag di inginkan

    terdapat 3 cara untuk mengakhir absen :

    cara 1:

  • buka jadawal absen
  • klik tutup absen untuk hari ini

    • cara 2:

  • buka control panel
  • ketik perintah "close"

    • cara 3:

  • buka control panel
  • ketik perintah "auto"

  • buka menu rekap harian bulana matuapun tahunan
  • lalu klik tombol di atas table untuk mencetak rekap
