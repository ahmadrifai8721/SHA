Saya Hadir App Control Panel ( SHA CP)

Program ini di gunakan untuk mengontrol Program SHA.

Apa saja yang ada di SHA CP

  • menyalakan dan mematikan server
  • menyalakan dan mematikan maintenance server
  • mengatur pengakiran absen auto dan manual

    • Daftar perintah dan fungsinya

    No Perintah Fungsi Note
    0 exit keluar dari Control Panel
    1 run menjalankan atau mengaktifkan SHA server
    2 stop mematikan SHA server
    3 absen-status melihat jadwal auto Close absen selanjutnya
    4 close Mengakhiri absen saat itu juga seteleh absen di akhiri guru dan siswa tidak dapat absen lagi
    5 auto menjalankan system auto close absen setelah perintah ini di jalankan jangan tutup windows cmd nya
    6 cmd menjalankan perintah cmd ( seperti ipconfig, ping dll )
    7 install menginstall program
    8 remove menghapus program beserta databasenya
    9 maintenance masuk ke maintenance mode ketik on untuk mengaktifkan maintenance ketik off untuk menonaktifkan maintenance ketik back untuk keluar dari menu maintenance
    1 status cek status server
