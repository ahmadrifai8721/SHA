Cara instal Program SHA

setep 1

jalankan SHA.exe dan install ke C: (*Wajib di local disk C )

  • buka SHA.exe
  • tekan browse arahkan ke local disk C:
  • lalu tekan extract dan tunggu hingga proses extract selesai ( proses ini mungkin agak lama )
  • jika sudah selesai tutup SHA.exe

    • setep 2

    menjalankan control panel

  • buka direktor C:\SHA\
  • run as anministrator Contol Panel.cmd
  • tunggu hingga Contol Panel Terbuka jika sudah lanjut ke step 3

    • setep 3

    Proses instalasi SHApa

  • buka Control Panel
  • ketikan "install" (tanpa tanda petik )
  • tunggu hingga Proses instalasi selesai lalu tekan enter

    • setp 4
